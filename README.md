# Guide Prompt Engineering

Support pédagogique complet sur le **Prompt Engineering** pour étudiants CDA, débutants et développeurs web.

**Démo :** https://wazabi64000.github.io/guide-Prompt-Engineering./

Aucun framework, aucune bibliothèque externe, aucun CDN, aucun backend : ouvrez simplement `index.html` dans un navigateur.

## Démarrage rapide

```bash
# Ouvrir index.html, ou servir localement :
python3 -m http.server 8080
```

## Structure

```
├── index.html
├── css/          # variables, style, components, animations
├── js/           # app, navigation, clipboard, search, progress, storage
├── assets/
└── README.md
```

## Contenu

Introduction → 11 chapitres (prompts, bonnes pratiques, outils, concepts avancés, bibliothèque ≥ 50 prompts) → conclusion.

## Fonctionnalités

Menu latéral, progression de lecture, recherche, accordéons, onglets, copie presse-papiers, LocalStorage, animations, responsive, accessibilité ARIA.
